## Getting Started

1. Generate your app 
    ```shell
    $ npx parcel-react-starter your-app-name
    ```

2. Navigate to the directory for your newly generated app (`./your-app-name` for instance)
    ```shell
    $ cd your-app-name
    ```

3. Run a fresh install
    ```shell
    $ yarn
    ```

4. Start your app!
    ```shell
    $ yarn start
    ```
